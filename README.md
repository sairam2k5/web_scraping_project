# Book Store Web Scraping Project

## Overview

This project uses Python to scrape book information from the Books to Scrape website.

## Tools Used

* Python
* Requests
* BeautifulSoup
* Pandas
* Matplotlib

## Data Collected

* Book Title
* Price
* Availability

## Dataset

* Total Records: 1000 books
* Source: Books to Scrape

## Analysis Results

* Average Price: 35.07
* Highest Price: 59.99
* Lowest Price: 10.00

## Files

* multi_page_scraper.py
* analyze_book.py
* all_books.csv
* all_books_cleaned.csv

## Conclusion

The project demonstrates web scraping, data cleaning, and data analysis using Python.
